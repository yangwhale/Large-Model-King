# 风格指南

## 叙事原则

### 总体风格定位
参考无罪《通天之路》的"轻松流"写法：
- **用说人话的方式写修仙**。不端着，不装文艺，文字要像跟朋友聊天一样自然。
- **小人物看大世界**。Chris Yang 就是 AI 修仙界的"魏索"——一个筑基期的小散修，在大佬们的天地巨变中夹缝求生、冷眼旁观。他不是英雄，是一个带着 Google 工牌的普通打工人，恰好站在了风暴中心。
- **技术硬核 + 叙事轻松**。技术细节必须准确，但讲述方式要轻松。像《通天之路》把复杂的修炼体系写得老少皆宜一样，我们要把 Transformer、MoE、RLHF 写得让不懂技术的人也能看热闹。
- **幽默是核心调味料，不是点缀**。自嘲、吐槽、调侃都可以，战斗写热血，日常写搞笑，两者自然切换。

### 视角
- **主线**: 第三人称全知视角。像修仙小说一样俯瞰整个世界的变化。
- **插叙**: 偶尔切入第一人称 Chris Yang（观察者/穿越者）视角。不端 Google 专家的架子，带着自嘲和真实感。像个在大能斗法现场偷偷发朋友圈的围观群众。
- **节奏**: 大事件用浓墨重彩（一章一个大事件）。过渡期用快速蒙太奇（时间线概述）。

### 基调
- **正文**: 白话为主，偶尔点缀半文半白的修仙腔调。不要通篇古风——现代读者读着累。关键场景（突破、陨落、大战）可以用古风拔高气氛，日常叙事用大白话。
- **战斗场景**: benchmark 比试 = 斗法。模型发布 = 突破境界。开源 = 传道。写战斗要有画面感和节奏感，参考《通天之路》的热血打斗——简洁、快、爽。
- **幽默感**: 无罪式的轻松感：角色可以有性格缺陷和搞笑时刻。Sam Altman 可以被写成"修仙界最不会修炼但最会搞钱的掌门人"。Jensen Huang 可以被写成"每次发布会都要穿皮衣装逼的灵石教主"。人物有烟火气才可爱。
- **对话**: 角色说话要有个性，不要都是一个腔调。技术宅说话像技术宅，商人说话像商人，学者说话像学者。可以用现代口语，不必强行古风。

## 文笔要求（参考《通天之路》）

### 必须做到
1. **读起来顺畅**：每一句话读出来都不拗口。写完自己念一遍，念不顺就改。
2. **节奏明快**：短句为主，长句偶尔穿插。不要一段话从头到尾不换气。
3. **画面感强**：写 GPU 集群就要让人"看到"那些闪着绿光的机柜，写训练就要让人"感受到"数据像洪水一样涌入模型。
4. **角色有温度**：每个人物出场要有记忆点。Ilya 的沉默、Sam 的笑容、梁文锋的冷峻——一两个细节就够了。
5. **每章都有"爽点"**：或者是技术突破的震撼，或者是商业博弈的反转，或者是人物命运的转折。不要写流水账。

### 绝对不要
1. 不要写又长又无聊的环境描写。
2. 不要堆砌修仙词汇装腔作势。
3. 不要让所有角色说一样的话。
4. 不要为了搞笑而搞笑——幽默要自然地长在情节里。
5. 不要连着三段没有对话或动作。

## 章节结构模板

每章大约 3000-5000 字。结构灵活，不必死板套用，但一般包含：

```
# 第 X 章：章名（要有吸引力，可以幽默、可以悬疑）

[开场白 — 一两句修仙腔调点题，简洁有力，不要写成散文]

[主体故事 — 以人物为中心展开。对话推动情节，动作推动节奏。]

[技术解析 — 融入故事中自然展现，不要单独拎出来像写教科书。
            用修仙术语包装但括号注释真名，让懂技术的会心一笑。]

[章末 — 要么留悬念，要么留余韵。让人想翻下一章。]

---
> 旁白（Chris 视角）：[放飞自我的吐槽和感悟。可以用现代梗、互联网黑话。
>                     这里是全书最"出戏"但最有趣的部分。]
```

## 写作禁忌

1. **不歪曲事实**: 技术细节必须准确。时间线必须正确。可以用修仙包装但不能编造不存在的技术。
2. **不贬低任何人**: 每个人物都有闪光点和缺点，但不恶意丑化。可以调侃但不侮辱。写 Yann LeCun 爱吵架是调侃，写他是小丑就过分了。
3. **不过度中二**: 修仙元素是调味料不是主菜。技术洞察才是核心价值。一旦发现自己写的东西连自己都觉得矫情，立刻删掉重写。
4. **不泄露私密**: 不写未公开的内部信息。只基于公开论文、新闻、产品发布。
5. **不写流水账**: 每一段都要有存在的理由——推动情节、塑造人物、或者埋伏笔。三者都不占就删掉。
6. **不学术化**: 这是网络小说不是论文综述。技术解释要像跟朋友喝酒时聊天，不像在做学术报告。
