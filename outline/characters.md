# 人物谱

## 天地造化者（互联网先驱）

他们不是修炼者，但他们铸造了灵脉、开辟了灵材矿藏。没有他们，修仙界不会存在。

| 真名 | 道号 | 门派 | 境界 | 简介 |
|------|------|------|------|------|
| Tim Berners-Lee | 灵脉祖师 | CERN / W3C | 化神期 | 万维网的发明者。他铸造了第一条真正意义上的灵脉。上司在他的提案上写了「模糊但令人兴奋」。他把灵脉设计成开放的——任何人都可以接入，任何人都可以贡献灵材。这个决定改变了一切。 |
| Larry Page & Sergey Brin | 灵材双圣 | Google 神殿 | 大乘期 | 斯坦福车库里写出了 PageRank。Google 搜索不只是灵材探测术——它索引了天下所有灵材，让修炼者能在万亿灵材中精准找到需要的那一份。Google 后来成为修仙界最强门派之一，因为他们从第一天起就在做"索引天下"这件事。 |
| 李飞飞 | 灵材女仙 | Stanford / Google | 化神期 | ImageNet 的缔造者。带领团队标注了 1400 万张图片。这不是普通灵材——这是经过人工精炼的天材地宝。2012 年 AlexNet 靠这份灵材一鸣惊人。后来所有计算机视觉的修炼者都受益于她的贡献。 |
| Linus Torvalds | 散修盟主 | Linux / Git | 化神期 | Linux 内核和 Git 的创造者。他证明了散修联盟可以炼出不输大门派的法器。Linux 跑在几乎所有训练集群上，Git 让全世界的修炼者可以协作开发功法。脾气暴躁，经常骂人。 |
