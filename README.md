# 大模王

> 一部以大模型时代为背景的修仙小说

当深度学习的灵气在 2012 年觉醒，修炼者们发现了一条通往通用智能的大道。从 AlexNet 的混沌初开，到 Transformer 的注意力觉醒，从 ChatGPT 引发的万界震动，到 DeepSeek R1 的推理觉醒——每一次突破都是一场修仙界的天地巨变。

这是一个关于算力、算法和野心的故事。关于那些试图让机器学会思考的人。
