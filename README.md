# 大模王

> 一部以大模型时代为背景的修仙小说。大模型不是丹药，是**神兽**。

当深度学习的灵气在 2012 年觉醒，修炼者们发现了一条通往通用智能的大道。他们孵化神兽、驯服神兽、让神兽互相竞逐。从 AlexNet 的混沌初开，到 Transformer 的注意力法典降世，从 ChatGPT 引发的天下震动，到 DeepSeek R1 用群兽竞逐法逆天改命——每一次突破都是修仙界的天地巨变。

这是一个关于灵石（算力）、灵材（数据）、功法（算法）和野心的故事。关于那些试图孵化出有智慧的神兽的人。

## 核心设定

- **神兽** = 大模型。有灵性、有能力、有脾气的活物。每个门派的镇派之宝。
- **灵石** = GPU/TPU 算力。修炼的能量来源。
- **灵材** = 训练数据。互联网上的文字、图片、视频都是天材地宝。
- **灵脉** = 互联网基础设施。连接一切、运输一切的管道。
- **育兽法阵** = 深度学习框架（PyTorch/JAX）。没有法阵，养不出神兽。
- **御兽之法** = RL 对齐方法（PPO/DPO/GRPO）。驯服神兽的各种流派。

## 风格

- 参考无罪《通天之路》的轻松流写法：**用说人话的方式写修仙**
- 真实的技术细节和历史事件，修仙术语包装但不歪曲事实
- 每一个「神兽」「灵石」「功法」「法阵」都对应真实的 AI 技术
- 懂技术的人看门道，不懂技术的人看热闹
